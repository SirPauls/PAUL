# Security Policy

## Supported Versions

The following versions of PAUL (Pattern & Asset UI Library) are currently being supported with security updates:

| Version | Status | Supported          |
| ------- | ------ | ------------------ |
| 0.0.x   | Pre-Alpha | :white_check_mark: |

## Reporting a Vulnerability

We take the security of our software seriously. If you find a vulnerability, please report it to us immediately.

**To report a security vulnerability:**

Please open an issue in our GitHub repository. 

Since we do not currently have a dedicated private security email, please exercise caution when reporting sensitive vulnerabilities. If possible, use GitHub's **Private Vulnerability Reporting** feature if it is enabled on the repository.

Please include the following details in your report:

*   Description of the vulnerability
*   Steps to reproduce
*   Potential impact

We will acknowledge your report as soon as possible and provide an estimated timeline for a fix.
